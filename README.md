# LIFX-API-RS

## Description

A synchronous + asynchronous library for communicating with the LIFX-API. 

## Supported API Methods:
* List Lights
* Set State
* Clean

## How to use library

Add the following line to your cargo.toml:
```
lifx-api-rs = "0.1.0"
```

Example:
```rust
extern crate payup;

fn main() {


}

```
